About The Project

Traditional attendance systems are time-consuming and prone to proxy marking. EduTrack AI automates this process using Face Recognition technology. Additionally, it analyzes historical data to predict future grades, helping educators identify at-risk students early.

Key Features

Face Recognition

Automatically identifies students via webcam in real-time using OpenCV and Haar Cascades.

Performance Prediction

Uses Regression models to predict final grades based on attendance and past performance.

Database Integration

Stores attendance logs and student data securely using SQLite/MySQL.

Report Generation

Exports attendance and prediction reports to Excel/PDF for easy administration.

How It Works

1

Input
